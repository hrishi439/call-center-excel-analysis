# Call Centre Data Analysis using Excel

This project demonstrates basic data analysis on call centre data using Microsoft Excel.  
The goal is to answer common business and operational questions using Excel features.

---

## Dataset Overview

The dataset contains call centre records including:

- Customer ID
- Customer Name
- Call Date (Day, Month, Year)
- Call Duration
- Customer Satisfaction Rating
- Customer Service Representative
- Sales / Call Metrics

---

## Business Questions Answered

1. How many calls are received from each customer?
2. How satisfied are our customers?
3. Who are the top 10 customers?
4. Who are the top 10 customers for a specific representative?
5. What is the call duration distribution?
6. How busy was the call centre in 2023?
7. What is the Year-To-Date (YTD) performance?
8. Which days of the week are the busiest?
9. Is there a relationship between call duration and customer satisfaction?
10. Should additional staff be hired in specific months?

---

## Analysis Techniques Used

- Pivot Tables
- Pivot Charts
- Filters and Slicers
- Conditional Formatting
- Grouping and Bucketing
- Calculated Fields
- Trend and Time Series Analysis
- Scatter Plot Analysis

---

## Tools Used

- Microsoft Excel

---

## Project Outcome

This analysis helps understand customer behavior, service quality, operational workload, 
and staffing requirements in a call centre environment.
